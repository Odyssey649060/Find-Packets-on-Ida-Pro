How to Find Packets on IDA Pro

This is a simple and detailed guide on how to find packets in IDA Pro for your Brawl Stars server.

Go to the Step-by-Step Guide

Requirements

Before we start, you'll need the following tools:

  • IDA Pro: Download here
  • The libg file with debug symbols
  • The libg.so of your version of Brawl Stars (we'll use V26 for this example)
  • A custom IDA script to fix the V26 debug symbols

To apply the script, go to IDA Pro → File > Script File and select it after the autoanalysis completes.

My First Packet

Let's start with something simple.

10121: UnlockAccountMessage — this is the ID and name of the packet.

Open the two files in IDA Pro and wait for the autoanalysis to finish. Then in the lib with debug symbols (V36), run the aforementioned script.

Understanding the Structure

It is possible that in your version there are more subs, so we need to identify what each sub_ is.

sub_3D1124 = writeLong
sub_F1B90  = writeString
sub_F1B90  = writeString
sub_F1B90  = writeString

The subnames apply to all packets in this lib. In another version of Brawl Stars, the subname will likely be different.

Tip for Beginners

I recommend that beginners start by porting packets from one version to another. For example, try porting TeamSetLocationMessage from V15 to V22.
