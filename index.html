How to Find Packets on IDA Pro

This is a simple and detailed guide on how to find packets in IDA Pro for your Brawl Stars server.

Go to the Step-by-Step Guide

Requirements

Okay, before we start, you'll need the following tools:

  • IDA Pro: Download here
  • The libg file with debug symbols
  • The libg.so of your version of Brawl Stars (in this example, version V26)
  • A custom IDA script to fix V26 debug symbols

To apply the script: open IDA Pro, go to File > Script File, and select the script.

⚡ Note: Run this script after IDA completes self-analysis.

Starting IDA

When you open the file for the first time, you’ll see the analysis progress. Wait until it completes and looks clean.

My First Packet

Let’s start with something simple.

10121: UnlockAccountMessage

This is the ID and name of the packet.

Steps

  1. Open both the libg.so and the libg with debug symbols in IDA Pro.
  2. Wait for autoanalysis to finish.
  3. In the debug version (V36), run the script as described above.
  4. Press ALT + i and look for UnlockAccountMessage.
  5. Double-click the result.
  6. Press X to find cross-references.
  7. Go to the function _ZN20UnlockAccountMessage6encodeEv.
  8. Double-click it and then press F5 to decompile.

Understanding the Structure

You will now see the packet structure. In this case:

sub_3D1124 = writeLong
sub_F1B90  = writeString
sub_F1B90  = writeString
sub_F1B90  = writeString

These function names (like sub_3D1124) refer to internal functions such as writing data. They can change between versions.

Tip for Beginners

If you're starting out, it's a good idea to try porting packets from one version to another.

For example, try porting TeamSetLocationMessage from V15 to V22 to understand how the structures evolve.
